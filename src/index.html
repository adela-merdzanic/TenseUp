How it works

  • Questions are pulled from the selected topics and shuffled together, just like the real exam.
  • Once you answer a question correctly, it's removed from future runs - so each new session only quizzes you on what you haven't mastered yet.
  • Use "Reset progress" any time to bring back the full question pool and start over.
